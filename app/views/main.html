Compact inline SVG symbols of flags to use in single page html5 apps!

Code to paste:
<div style="display:none"> <svg xmlns="http://www.w3.org/2000/svg"> {{co}} </svg> </div> Select flags...
Flags:

How to use:

Select the flags you want and then paste the code from above after opening the <head> tag of your html5 page. Then you can use the flags by referencing their id: <svg width="40px" height="30px"> <use width="40px" height="30px" xlink:href="#flag-id" /> </svg>

So to use, for example the German flag with width 80, we can use the following code: <svg width="80px" height="60px"> <use width="80px" height="60px" xlink:href="#flag-de" /> </svg>
