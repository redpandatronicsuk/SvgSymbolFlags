SvgSymbolFlags


Compact inline SVG symbols of flags to use in single page html5 apps!

Select Flags:

{{flag.id}}
Code to paste:
<div style="display:none"> <svg xmlns="http://www.w3.org/2000/svg"> {{co}} </svg> </div> Select flags...
Flags:
download

How to use:

Select the flags you want and then paste the code from above after opening the <head> tag of your html5 page. Then you can use the flags by referencing their id: <svg width="40px" height="30px"> <use width="40px" height="30px" xlink:href="#flag-id" /> </svg>

So to use, for example the German flag with width 80, we can use the following code: <svg width="80px" height="60px"> <use width="80px" height="60px" xlink:href="#flag-de" /> </svg>

Alternatively:

Once you have selected all the flags you want to include, hit the download button and check out the source code of the generated HTML file. NOTE: You might have to save the file to your computer first (File -> Save page as..., Cmnd/Ctrl + s, or wahtever) and re-open it, before the flags show up in your browser.

How to contribute:

The aim of this project is to provide SVG flag symbols in their smallest size possible. This means that flags should be with the fewest SVG tags, parameters and parameter values as possible. All flags should have 4:3 aspect ratio and some symbols on the flag can be sacrificed in favour of smaller code size. If you have any SVG flags you would like to contribute, please see email us at: info@redpandatronics.com or see further instructions on the github repository.
